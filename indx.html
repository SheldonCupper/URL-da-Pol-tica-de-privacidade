Política de Privacidade

Última atualização: 24 de abril de 2026

Gabriel Alves de Oliveira Brito construiu o aplicativo Planner Zilla como um aplicativo Gratuito. Este SERVIÇO é fornecido por Gabriel Alves de Oliveira Brito sem custo e destina-se a ser utilizado no estado em que se encontra.

Esta página é usada para informar os visitantes sobre as políticas de coleta, uso e divulgação de Informações Pessoais, caso alguém decida usar o Serviço.

Coleta e Uso de Dados

O Planner Zilla foi desenvolvido com a sua privacidade em mente. Todos os dados relacionados às suas tarefas, rotinas, hábitos e planejamento inseridos no aplicativo são salvos localmente no seu próprio dispositivo através de tecnologias de armazenamento local. Nós não transmitimos, armazenamos em servidores externos, nem temos acesso aos seus dados pessoais de produtividade.

Dados de Registro (Log Data)

Sempre que você usar o Serviço, em caso de erro no aplicativo, dados e informações podem ser coletados (através de produtos de terceiros, como a Apple) no seu dispositivo, chamados de Dados de Registro. Esses dados podem incluir informações como o endereço IP, nome do dispositivo, versão do sistema operacional e estatísticas de uso para fins exclusivos de correção de erros e melhoria da estabilidade.

Segurança

Como os dados principais do aplicativo ficam armazenados no seu aparelho, a segurança do acesso físico ao seu dispositivo é de sua responsabilidade. Recomendamos o uso das proteções nativas do sistema operacional (como FaceID, TouchID ou senha) para garantir a privacidade dos seus dados locais.

Mudanças nesta Política de Privacidade

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos a revisão desta página para verificar quaisquer alterações. Notificaremos você sobre mudanças publicando a nova versão nesta página.
